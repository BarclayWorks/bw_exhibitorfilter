# BW Exhibitor Filter

Joomla system plugin for filtering and managing exhibitor displays.

## Description
System plugin that provides advanced filtering capabilities for exhibitor listings, enabling dynamic content filtering based on various criteria.

## Features
- Dynamic exhibitor filtering
- Customisable filter criteria
- Integration with Joomla content system
- Compatible with Joomla 4.x+
